# CS 61B Project 3

## Project Description
CS 61B Project 3 focuses on data structures and algorithms, giving students practical experience with graph processing and implementation of algorithms.

## Technologies
- Java
- Graph Data Structures
- Algorithms for Graph Traversal
- JUnit for testing

## Key Features
- Implementation of various graph algorithms such as Depth-First Search (DFS) and Breadth-First Search (BFS).
- Ability to visualize graphs and their traversal paths.
- Testing and validation of graph implementations using JUnit.
